Stelt

This is an web application I personally created. It's a style focused app centered around celebreties and their outfits. The main goal of the app is to allow users to emulate the styles of their favorite celebs at affordable prices. The inspiratoin behind this idea came from seeing LeBron james pull up to last years NBA Finals clad in a $45,000 suit that replaced the more typical pair of slacks for a pair of uhh...suit shorts. anyways, I thought to myself there has to be a affordable way for people to dress like the stars. On my app you'll find pictures of various famous people, and there outfits. When you select an outfit you are brought to a page that displays pictures of products that look simular to each individual item the celebrity is wearing. clicking on that picture pulls up the site the item is hosted on, and the prices are affordable for us normals. that's the main feature of the application. Just because you don't possess the looks the bod or the hair; it doesn't mean you can't copy what they wear

The backend for my app was built using Rails along with Postgresql database containing several models and joint tables, and the frontend for this project was Vue.JS based. It aslo includes the tumblr-Api which under my original intention was meant to be used in conjunction with another Api named Thread Genius. It was a style focused image recognition Api with built in catalogues with thousands of products from places like shopstyle etc. as well as fashion blogs, allowing you to take an image of an outfit and have it pull up products that look simular. it woild have been very cool but unfortunately they were acquired recently leaving their Api unusable. So now Plan B when I get the chance is to add Apis from some fashion brands in order to pull up their catalog in app to make outfit creation even easier

As far as other ongoing projects, I'm currently working with some guys building a 2-D Platform game on Unity as a fun side project. it's been alot of fun so far, and I've been working on writing the C# scripts for the enemy AI, and also things like the game physics. Hopefully we can get out a playable level out soon
