Stelt

This is a web application I personally created. It's a style focused app centered around celebrities and their outfits. The main goal of the app is to allow users to emulate the styles of their favorite celebs at affordable prices. The inspiration behind this idea came from seeing LeBron James pull up to last year’s NBA Finals clad in a $45,000 suit that replaced the more typical pair of slacks for a pair of uhm...suit shorts. Anyways, I thought to myself there must be an affordable way for people to dress like the stars. On my app you'll find pictures of various famous people, and their outfits. When you select an outfit, you are brought to a page that displays pictures of products that look like each individual item the celebrity is wearing. Clicking on that picture pulls up the site the item is hosted on, and the prices are affordable for us Normals. That's the main feature of the application. Just because you don't possess the looks the bod or the hair doesn't mean you can't copy what they wear inexpensively.

The backend for my app was built using Rails along with Postgresql database containing several models and joint tables, and the frontend for this project was Vue.JS based. It also includes the tumblr-Api which under my original intention was meant to be used in conjunction with another Api from Thread Genius. Which was a style focused image recognition Api with built in catalogues with thousands of products from places like Shopstyle etc., as well as fashion blogs, allowing you to take an image of an outfit and have it search for products that look similar. The Thread Genius Api would have been very cool but unfortunately, they were acquired by another company recently leaving their Api unusable. So now my plan B for my app is to add APIs from some fashion brands in order to pull up their catalog in the app to make outfit creation even easier.

As far as other ongoing projects, I'm currently working with partners building a 2-D Platform game on Unity as an enjoyable side project. It's been a lot of fun so far. Specifically, I've been working on writing the C# scripts for the enemy AI, and also things like the game physics. Hopefully we will launch a playable version on here soon.
